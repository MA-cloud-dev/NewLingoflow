# 相关技能 (Skills)


*   **计算机基础**：
    *   了解操作系统基本原理及 Linux 常用命令，具备基本的服务器部署与运维能力。

*   **Java 核心**：
    *   熟练掌握 **Java** 基础知识，包括面向对象编程、集合框架（HashMap, ArrayList）、多线程并发编程（Thread, ExecutorService）及 Java 8 新特性（Stream API, Lambda）。
    *   了解 **JVM** 内存模型及基础的垃圾回收机制，具备基本的代码排查与优化能力。

*   **开发框架**：
    *   熟练使用 **Spring Boot 3.x** 进行快速开发，理解其自动配置原理与 Starter 机制。
    *   熟悉 **Spring MVC** 请求处理流程，熟练运用 Spring Security / JWT 进行安全认证与权限管理。
    *   熟练使用 **MyBatis / MyBatis-Plus** ORM 框架进行数据持久化操作，具备复杂 SQL 编写能力。

*   **数据库**：
    *   熟练掌握 **MySQL** 关系型数据库，了解 InnoDB 存储引擎，具备各种连表查询（Join）、索引优化及事务处理（ACID）经验。
    *   了解并使用 **Redis** 非关系型数据库，了解 String, List, Set, Hash, ZSet 等数据结构，具有在项目中使用 Redis 实现缓存（Caching）及队列（Queue）的实战经验。

*   **前端技术**：
    *   了解并使用过 **HTML5 / CSS3 / JavaScript (ES6+)**，熟悉现代前端开发流程。
    *   了解并使用过 **Vue 3** 全家桶（Vue Router, Pinia）及 **TypeScript** 进行组件化开发。
    *   了解并使用过 **Tailwind CSS** 及 Element Plus 等 UI 框架进行响应式界面设计。
    *   熟悉使用 **Node.js** 进行前端开发，了解 npm 包管理工具。
    *   熟悉使用 **AI 工具 (Cursor / Copilot)** 进行辅助开发，显著提升编码与调试效率。

*   **Python 开发**：
    *   熟悉 **Django** Web 框架，具备后端开发经验；了解 **Flask** 轻量级框架。
    *   熟悉 **网络爬虫** 技术，能够进行数据采集与清洗。
    *   了解 **数据分析** 基础（Pandas, NumPy）及 **神经网络** 基础。

*   **工具与中间件**：
    *   熟练使用 **Maven** 进行项目构建与依赖管理。
    *   熟练使用 **Git** 进行版本控制与团队协作（Branch, Merge, Rebase）。
    *   熟悉 **Docker** 容器化技术，了解基本的镜像构建与容器部署。
    *   熟悉 **Nginx** 反向代理,负载均衡到后台服务器。

*   **新兴技术 (AI Integration)**：
    *   具备 **LLM (大语言模型)** 应用开发经验，熟悉 Prompt Engineering（提示词工程），能够通过 API 集成（如 SiliconFlow）实现业务逻辑与 AI 能力的结合。
