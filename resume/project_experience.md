# 项目经历

## LingoFlow AI 沉浸式英语学习平台

**项目描述**：
LingoFlow 是一个基于大语言模型（LLM）驱动的垂直领域英语单词记忆应用。项目旨在解决传统死记硬背效率低下的痛点，通过 AI 根据用户生词本实时生成具有上下文语境的短文（Contextual Learning），并结合阅读理解、智能造句批改以及 SM-2 记忆算法，为用户提供“学-练-测-复习”的完整闭环体验。

**技术栈**：
Spring Boot 3 + MySQL 8.0 + Redis + MyBatis-Plus + Vue 3 + TypeScript + Tailwind CSS + SiliconFlow AI API + JWT

**工作内容 / 个人职责**：
*   **0-1 架构设计与搭建**：主导后端 Spring Boot ,使用JWT进行用户认证；完成了核心数据库模型（ERD）设计，包含词典、学习会话、复习记录等 10+ 张核心表。
*   **AI 核心业务与 Prompt 工程**：深度集成 SiliconFlow 大模型 API，设计并优化了一套 Prompt 模板系统，能够根据用户的目标词汇、难度偏好（Theme/Difficulty）动态生成高质量的英语短文、阅读理解题及造句纠错反馈，显著提升了内容的丰富度和学习趣味性。
*   **复习算法与高并发缓存**：实现了基于 SM-2（SuperMemo-2）的间隔重复复习算法。为解决复杂查询性能问题，引入 Redis 缓存层，利用 `List` 和 `Hash` 结构缓存用户的每日复习队列和实时学习状态（LearningState），大幅降低了 MySQL 的查询压力，保证了学习流程的流畅性。
*   **AI 驱动前端开发与联调**：个人驱动 Gemini 辅助高效开发 Vue 3 前端核心模块，并成功完成与后端的全量接口对接。采用 TypeScript 保证类型安全，主导了 "Editor's Focus" 风格的 UI 重构，使用 Tailwind CSS 实现极简深色模式界面，TTS 朗读等交互细节。
*   **数据统计与可视化**：设计实现了用户学习数据统计模块，基于 ECharts 开发了“周学习曲线”和“记忆持久度”仪表盘，通过 SQL 聚合查询与前端图表联动，直观展示学习成果。
**项目成果**：
*   成功构建了一个 AI 驱动的 MVP 产品，实现了从“选词”到“AI 生成文章学习”再到“智能复习”的完整业务闭环。
*   通过 Redis 缓存策略，将学习页面的状态加载速度提升了 50% 以上，有效支撑了复杂的交互逻辑。
